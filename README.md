#project objective
This Power BI report analyzes website performance using key metrics such as Pageviews, Bounce Rate, Exit Rate, and Unique Pageviews.
The dashboard is designed in 2 pages, providing both a high-level overview and a detailed breakdown of page-level performance.
#data set
- <a href ="https://github.com/kanika-015/Data_analysis_dashboard/blob/main/Website%20Analytics.xlsx">Dataset</a>
#Questions (KPIs)
- How many total pageviews did the website receive?
- How many unique pageviews were generated across all pages?
- Which quarter recorded the highest website traffic?
- Which month performed best in terms of pageviews?
- Which page received the maximum pageviews?
- What is the overall average bounce rate of the website?
- What is the overall average exit rate across all pages?
- How do pageviews vary by Yearly, Quarterly, and Monthly filters?
- Which pages show the highest bounce rate and need improvement?
- How is the traffic distributed across all quarters (Q1, Q2, Q3, Q4)?
- How do pageviews compare across different pages (Page 1, Page 2, etc.)?
- What percentage of total pageviews comes from each quarter?
- What is the minimum, maximum, and average pageview count?
- How do user engagement metrics (Pageviews vs Unique Pageviews) differ?
- Which pages have high exit rates indicating content drop-offs?
